# CloudSyncApp
CloudSync Android Application README




What is it?
---------------


CloudSync is an android app aimed at accessing your cloud storage from various cloud storage providers and itâ€™s sharing easier by providing a single platform to do so. The main functionality of the app is presenting the services of Google Drive, Dropbox, One Drive, etc. from a unified cloud file manager. The app achieves rich user interface for the users by using material design. Single panel access using navigation drawer and sliding tabs is another powerful feature of the application that makes it quick and easy to use.




Current Feature(s)
------------------------




-ONE-STOP ACCESS for all your cloud storage.
-Tabbed view according to the type of a file.
-Single Panel representation of files from different cloud services.
-File management : Download and delete files and folders.
-File preview.
